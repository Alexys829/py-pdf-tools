# PyPDF Tools

Universal PDF toolkit with Material Design GUI.

## Features

| Tool | Description |
|------|-------------|
| **Editor** | View page thumbnails, add/remove/reorder pages from multiple PDFs |
| **Merge** | Combine multiple PDFs into one with reordering |
| **Split** | Split into single pages or extract a page range |
| **Rotate** | Rotate 90/180/270 - all, odd, even, or custom pages |
| **Extract Images** | Save all embedded images as PNG or JPG |
| **Images to PDF** | Combine images into a single PDF |
| **Compress** | Reduce file size with adjustable image quality |
| **Password** | Encrypt (AES-256) or decrypt PDFs |
| **Page Numbers** | Add customizable page numbering |
| **Watermark** | Add text watermark (File menu) |

## Installation

```bash
git clone https://github.com/Alexys829/py-pdf-tools.git
cd py-pdf-tools
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```
